# POS Gas Mobile (Android)

MVP en Kotlin + Jetpack Compose para tablet, conectado a Google Sheets vía Apps Script.

## Flujo
1. Deploy de `pc/MobileApi_BE.js` como Web App.
2. Pegar URL `/exec` en `MainActivity.kt`.
3. Correr app en Android Studio.

## Estética
Se usa tema oscuro con acento naranja `#FF5500`, alineado al estilo actual del sistema web.
